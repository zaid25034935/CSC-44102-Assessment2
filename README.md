## CSC-44102 Assessment 2: Guess the Number Game

This is a simple text-based "Guess the Number" game implemented in Python.

### Features
- **Core Gameplay:** The user attempts to guess a randomly generated number.
- **Difficulty Selection:** The user can select a difficulty level (Easy, Medium, or Hard), which determines the range of the secret number (up to 50, 100, or 200, respectively).

### How to Run
1. Ensure you have Python installed.
2. Run the game from your terminal:
   ```bash
   python guess_game.py
